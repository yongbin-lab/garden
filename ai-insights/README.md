# AI Insights

업무와 사이드 프로젝트에서 AI를 활용하며 얻은 실전 인사이트를 정리합니다.

## Categories

### Workflows
AI를 활용한 실제 업무 워크플로우 사례와 개선 과정을 기록합니다.

### Prompts
반복적으로 사용하는 프롬프트 패턴과 그 효과를 정리합니다.

## Writing Format

각 글은 아래 구조를 따릅니다:

```markdown
# 제목

- **상황**: 어떤 문제/과제가 있었는지
- **활용**: AI를 어떻게 적용했는지
- **결과**: 어떤 변화가 있었는지
- **배운 점**: 다음에 활용할 인사이트
```
