## Structure

| Directory | Description |
|-----------|-------------|
| [`ai-insights/`](./ai-insights) | AI 활용 인사이트 & 워크플로우 |
| [`plugins/`](./plugins) | 직접 만들거나 큐레이션한 플러그인 & 도구 |
| [`books/`](./books) | 책 요약 & 핵심 인사이트 |
| [`til/`](./til) | Today I Learned - 짧은 학습 기록 |
| [`projects/`](./projects) | 사이드 프로젝트 기획 & 아이디어 |

## How I Use This

- 새로 배운 것은 `til/`에 빠르게 기록
- 깊이 있는 내용은 각 카테고리 디렉토리에 정리
- 초안은 private repo에서 작성 후 공개 준비가 되면 이곳으로 이동
